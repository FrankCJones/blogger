== README

This is a Rails app similar to Blogger. It was created as an assignment from
the Web Development Frameworks section of The Odin Project
(http://www.theodinproject.com/web-development-101/ruby-on-rails) and the
instructions come from Jumpstart Lab
(http://tutorials.jumpstartlab.com/projects/blogger.html).

This application is currently deployed on Heroku at:

https://serene-scrubland-50435.herokuapp.com/
